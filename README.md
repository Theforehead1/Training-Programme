# Workout Tracker

A static GitHub Pages workout tracker for the 4-day shoulder and arm-focused split.

## Files
- `index.html`
- `styles.css`
- `manifest.json`
- `sw.js`
- `icon.svg`

## GitHub Pages setup
1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to Settings > Pages.
4. Select Deploy from a branch.
5. Choose `main` and `/root`.
6. Open the GitHub Pages link.
7. On iPhone, open the link in Safari, tap Share, then Add to Home Screen.

## Notes
Your workout data is stored locally in your browser. Use Export regularly as a backup.
