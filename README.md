# Workout Tracker

A simple static workout tracker for a 4 day shoulder and arm focused split.

## Features

- Week selector: Week 1, Week 2, etc.
- Workout selector for all 4 training days
- Input weight and reps for every working set
- Shows previous week's weight and reps for the same workout and exercise
- Notes field per exercise
- Copy previous week into the current week
- Export/import your data as JSON
- Works as a static GitHub Pages site
- Can be saved to your phone home screen

## How to host on GitHub Pages

1. Create a new GitHub repository.
2. Upload all files in this folder.
3. Go to **Settings > Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Save.
6. Open the GitHub Pages URL on your phone.
7. On iPhone Safari, tap **Share > Add to Home Screen**.

## Data note

The tracker saves data in your browser using localStorage. Use **Export data** occasionally as a backup.
